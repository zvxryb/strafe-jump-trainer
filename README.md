# Strafe Jump Trainer

![screenshot](./screenshot.png)

## Running

Run `wasm-pack build --target no-modules` from source root to build a new WASM binary, then change to the `utils` subdirectory and call `cargo run` to serve locally at 127.0.0.1:8080.
